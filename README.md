**Mouteur De Matching De Nom**

**Description**
**Mouteur De Matching De Nom** est une application Java qui permet de **comparer et identifier des correspondances entre des noms** provenant de différentes sources, comme des fichiers CSV ou des bases de données.  
Cette application est particulièrement utile pour **détecter les doublons** et **normaliser les données** dans des projets professionnels ou académiques.

 **Fonctionnalités**
- **Prétraitement et nettoyage des données** pour améliorer la qualité des correspondances.  
- **Génération automatique de candidats** pour faciliter la comparaison des noms.  
- Utilisation de **plusieurs algorithmes de similarité de chaînes**, notamment **Levenshtein** et **Jaro-Winkler**, pour identifier les correspondances les plus précises.  
- **Paramétrage flexible** permettant de sélectionner les résultats les plus pertinents.  

**Points forts**
- **Interface claire et intuitive** pour gérer facilement les noms.  
- **Algorithmes robustes** pour des comparaisons fiables et précises.  
- **Adaptable** à différents projets nécessitant la gestion et la déduplication de données textuelles.
